# Что такое Markdown?
Markdown - это облегченный язык разметки созданный с целью обозначения форматирования в простом тексте, с максимальным сохранением его читаемости человеком, и пригодный для машинного преобразования в языки для продвинутых публикаций <a href="http://example.com/" target="_blank">Hello, world!</a>
